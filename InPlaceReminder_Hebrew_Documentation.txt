תיעוד פרויקט InPlaceReminder - משתנים ופונקציות בעברית
===========================================================

1. מחלקה DB_OpenHelper - מנהל בסיס הנתונים
============================================

תיאור כללי:
מחלקה זו אחראית על כל פעולות בסיס הנתונים SQLite - יצירה, שדרוג, וניהול הנתונים.

משתנים (Constants):
- DATABASE_NAME: String = "in_place_reminder.db"
  תיאור: שם קובץ בסיס הנתונים שבו נשמרים כל הנתונים

- DATABASE_VERSION: int = 5
  תיאור: גרסה בסיס הנתונים הנוכחית (שדורגה 5 פעמים)

- TABLE_REMINDERS: String = "reminders"
  תיאור: שם טבלת התזכורות

- REMINDER_TITLE: String = "title"
  תיאור: שם עמודת הכותרת בטבלת התזכורות

- REMINDER_DESCRIPTION: String = "message"
  תיאור: שם עמודת התיאור בטבלת התזכורות

- PLACE_ID: String = "place_id"
  תיאור: שדה מפתח זר המתייחס למיקומים

פונקציות:

1. DB_OpenHelper(Context context)
   פרמטרים: context - הקשר של האפליקציה
   תיאור: בנאי המחלקה - יוצר או פותח חיבור לבסיס הנתונים

2. onCreate(SQLiteDatabase db): void
   פרמטרים: db - בסיס הנתונים
   תיאור: יוצרת את כל הטבלאות בבסיס הנתונים בפעם הראשונה
           - טבלת places (מיקומים)
           - טבלת reminders (תזכורות)
           - טבלת users (משתמשים)
           מכניסה משתמש ברירת מחדל "Admin" עם סיסמה

3. onUpgrade(SQLiteDatabase db, int oldVersion, int newVersion): void
   פרמטרים:
     - db - בסיס הנתונים
     - oldVersion - גרסה ישנה של בסיס הנתונים
     - newVersion - גרסה חדשה של בסיס הנתונים
   תיאור: מטפלת בשדרוג גרסאות בסיס הנתונים
           שומרת על תאימות לאחור עם נתונים קדומים

4. insertUser(String name, String passwordHash, byte[] picture): long
   פרמטרים:
     - name - שם המשתמש החדש
     - passwordHash - הסיסמה המוצפנת
     - picture - תמונת הפרופיל בבתים
   מחזירה: ID של המשתמש החדש או -1 אם הכניסה נכשלה
   תיאור: מכניסה משתמש חדש לטבלת המשתמשים

5. updateUserPicture(String name, byte[] picture): int
   פרמטרים:
     - name - שם המשתמש
     - picture - תמונה בבתים
   מחזירה: מספר השורות שהשתנו
   תיאור: מעדכנת את תמונת הפרופיל של משתמש לפי שם

6. updateUserPicture(long userId, byte[] picture): int
   פרמטרים:
     - userId - ID של המשתמש
     - picture - תמונה בבתים
   מחזירה: מספר השורות שהשתנו
   תיאור: מעדכנת את תמונת הפרופיל של משתמש לפי ID

7. userExists(String name): boolean
   פרמטרים: name - שם המשתמש לבדיקה
   מחזירה: true אם המשתמש קיים, false אם לא
   תיאור: בודקת האם שם משתמש כבר קיים בבסיס הנתונים

8. getUserPicture(Long id): byte[]
   פרמטרים: id - ID של המשתמש
   מחזירה: בתים של תמונת הפרופיל או null אם אין תמונה
   תיאור: מחזירה את תמונת הפרופיל של משתמש מסוים

9. getPrimaryUserName(): String (Nullable)
   מחזירה: שם המשתמש הראשון בבסיס או null
   תיאור: מחזירה את שם המשתמש הראשון בטבלה
           שימושית לאפליקציות שעובדות עם משתמש יחיד

10. verifyUserCredentials(String name, String candidatePassword): boolean
    פרמטרים:
      - name - שם המשתמש
      - candidatePassword - הסיסמה לבדיקה
    מחזירה: true אם הסיסמה נכונה, false אחרת
    תיאור: אומתת אם שם משתמש וסיסמה תואמים לנתונים בבסיס

11. getUserCursorByName(String name): Cursor (Nullable)
    פרמטרים: name - שם המשתמש
    מחזירה: Cursor עם נתוני המשתמש או null
    תיאור: מחזירה Cursor עם שורת משתמש (id, name, password_hash, picture)
            הקורא חייב לסגור את ה-Cursor כשסיים

12. getCircularBitmap(Bitmap src): Bitmap
    פרמטרים: src - התמונה המקורית
    מחזירה: התמונה שעובדה לעיגול
    תיאור: ממירה תמונה מלבנית לעיגול לתצוגת פרופיל מסודרת
